- notes
	RISC-V32I is the instruction set we will be using.
	some instruction types: Registers and Immediates

		Immediate ex.
			instruction destination, source1, immediate
			instruction destination, immediate(source1)

			"addi b, a, 5"
			"addi b, 5(a)"
		Regsiter ex.
			instruction destination, source1, source2
			"add a, a, b"
		
		- instruction	
			- the operation
		- destination
			- location of register
		- immediate
			- how far from current address to shift from
		- source1/source2
			- arguments of the instruction

	there are specific registers used for specific purposes.
	ex.
		0x, x2 the zero and Stack Pointer register


- memory notes
	- we are limited to 32 or less registers
	- we cannot use a value in memory for an operation. they must be loaded into a register

	lecturer vocab:
		RISC-V "Word"
		A "word" is 32 bits. The size of registers in RISC-V32I is 32 bits.

- machine encoding notes
	- the 32 bits can be used to
	identify the instruction
	identify the registers used
	identify any integer constant used in that instruction
	
	RISC-V32I is read little endian (least significant is read first)
	(1 is read first)
		0000001
